Design and Construction of a line following Robotic Car

Line following robot is an autonomous vehicle which detects black line to move over a bright surface. This project was constructed using the Arduino UNO microcontroller as the main component. It utilizes the IR Sensors to detect the black line on a bright surface. When the Infrared signal falls on the white surface, it gets reflected and it falls on the black surface/line. It also utilizes ultrasonic sensors for obstacle detection and avoidance.

COMPONENTS USED
Arduino UNO microcontroller | IR Sensors | Arduino IDE| Ultrasonic Sensors | SSG90 Servo Motor | 1860 Li-ion Battery| LEDs | 4WD Car Kit with Car chassis

A Real Time Hand Gesture-to-Voice Communication for Speech Impaired Individuals using Computer Vision

This project is a Real Time Hand Gesture-to-Voice Comunication that translates inputted hand gesture into a voice output and visual output on an LCD display. I achieved this by collecting dataset of hand gestures, training a model using the YOLO (You-Only-Look-Once) architecture and deploying the trained model on a Raspberry pi.

COMPONENTS USED
Raspberry pi | Raspberry Pi Camera Module | LCD Display| USB Speaker | Jumper Cables | Python| SD Card |
